Game Page - Acting

This is one of three views of the Game Page.

This view is shown when it's the user's turn. Here, the user is shown the hand history so far, their range for the current point in the hand, and they can then enter their ranges for their next action.

When the user completes their action, the page will become the Game Page - Waiting.
